* keyholder 制
** 所有茶舖客聚會免場地費
   但是如果活動有其他費用仍須支付
** 自由使用空間權
   同時也有維護這個空間的責任，包含使用後的復原，以及任何可能的損害的復原。
** 有teapook 發展基金如何使用的提案及投票權利
   在扣掉場地營運費用（房租，電費，水電，管理費等）剩下的資金將劃入「茶舖客發展資金」，
   keyholder 擁有提案與投票使用這筆金額的權利。
** 有teapook 發展如何制定的提案及投票權利
   經營方保留否決權。
